Oregon Solitaire

A simple solitaire card game about making poker hands and trying for the best score.

I wrote it in Visual Studio 2022 with C# for the back end and Windows Forms for the GUI. Just unzip and run the exe. Unfortunately, Windows Forms is a Microsoft product, so I can't guarantee it will run properly on a Mac.

You can view the git repository for this project here. If you don't trust my exe, please feel free to compile it from there.
