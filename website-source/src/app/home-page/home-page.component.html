About My Website

This homepage is angular app hosted on an AWS S3 bucket, and delivered to you through Amazon's CloudFront. I used Cloudfront, rather than the S3 static website hosting, because it allows me to deliver the site without making the S3 bucket public, which is required when using the built in website hosting.

I've written the code in Visual Studio Code. The code for the site is held in a Git repository on Github that can be viewed here. I've used Amazon's CodePipeline to deliver updated code to the S3 bucket. Code merged to the "deployment" branch of the repository is automatically delivered to the S3 bucket and goes live on my website.

I chose Amazon CodePipeline over other options, particularly GitHub's workflows. The CodePipeline was easy to set up, integrates directly with the S3 architecture, and doesn't require me to store additional private data and credentials on Github.
