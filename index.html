About My Website

This homepage is a static website hosted on an AWS S3 bucket, and delivered to you through Amazon's CloudFront. I chose Cloudfront because it allows me to deliver the site without making the S3 bucket public, which is required when using the built in website hosting functionality of an S3 bucket.

I've written the code in Visual Studio Code, as Visual Studio 2022 seemed like overkill for a simple project like this. The code for the site is held in a Git repository on Github that can be viewed here. I've used Amazon's CodePipeline to deliver updated code to the S3 bucket. Code merged to the "deployment" branch of the repository is automatically delivered to the S3 bucket and goes live on my website.

I chose Amazon CodePipeline over other options, particularly GitHub's workflows. The CodePipeline was easy to set up, integrates directly with the S3 architecture, and doesn't require me to store additional private data and credentials on Github.
